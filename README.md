# Zudio-Sales-Analysis
Zudio Sales Analysis Dashboard is an interactive Power BI dashboard built to analyze retail sales performance using business intelligence techniques. The dashboard provides insights into sales, profit, product categories, store types, states, and customer trends through dynamic visualizations, KPI cards, maps, and charts. 
# Zudio Sales Analysis Dashboard

## Overview
This project is an interactive Power BI dashboard created to analyze Zudio retail sales data. It transforms raw sales data into meaningful business insights using Power BI visualizations and DAX measures.

## Features
- 📊 KPI Cards (Total Sales, Total Profit, Average Profit)
- 📈 Monthly Sales Trend
- 🗺️ State-wise Sales Analysis
- 🍩 Profit by Category
- 🏬 Sales by Store Type
- 📉 Sales vs Profit Analysis
- 🎛️ Interactive Slicers and Filters
- 📌 Dynamic and User-Friendly Dashboard

## Tools Used
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Microsoft Excel

## Dashboard Insights
- Analyze monthly sales performance.
- Compare sales across different states.
- Identify profitable product categories.
- Evaluate store type performance.
- Track overall business KPIs.

## Skills Demonstrated
- Data Cleaning
- Data Modeling
- DAX Measures
- Data Visualization
- Business Intelligence
- Dashboard Design

## Project Goal
To create an interactive business intelligence dashboard that helps stakeholders monitor sales performance and make informed business decisions.
